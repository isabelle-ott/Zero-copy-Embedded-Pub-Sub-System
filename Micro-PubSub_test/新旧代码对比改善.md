## mps_free与mps_alloc搭配的错误

+ 原版问题：

Task A: free(ptr)  →  [CRITICAL] → 检查→释放 → [EXIT]
Task B:                                                    alloc() → 拿到同一块
Task A: free(ptr)  →  [CRITICAL] → 此时bitmap=0（已分配），ref=1 → 误杀！

+ 问题原因：测试阶段是单线程的没有出现，RTOS由于多线程会影响
+ 解决办法：claude->加入版本号

```c
//原版代码

taskEXIT_CRITICAL();
//上述作为保护

//上述作为判断条件，但是没有分区
```

## mps_add_ref的改变

```1c
MpsStatus_t mps_add_ref(MemPool_t *pool, MpsHandle_t *handle)
{
    if (!handle || !handle->ptr) {
        return MPS_ERR_INVALID;
    }

    int32_t idx = _ptr_to_idx(pool, handle->ptr);
    if (idx == -1 || idx != handle->idx) {
        return MPS_ERR_INVALID;
    }

    MpsStatus_t status = MPS_OK;
    taskENTER_CRITICAL();

    if (pool->generation[idx] != handle->generation) {
        status = MPS_ERR_STALE;             /* retain 时也要防 ABA */
    } else if ((pool->bitmap & (1u << idx)) == 0) {
    if (pool->ref_count[idx] < 255u) {
        pool->ref_count[idx]++;
    } else {
        status = MPS_ERR_REF_OVERFLOW;  // ✅ 明确返回错误
    }
        /* 原版 ref == 255 时静默丢弃，行为保留；可按需改为返回错误码 */
    } else {
        status = MPS_ERR_INVALID;
    }

    taskEXIT_CRITICAL();
    return status;
}
```

## mps_init

```1c
void mps_init(MemPool_t *pool)
{
    /*
     * TODO:
     *  设置 pool->bitmap，使得每个有效块的位都是 1（空闲）。
     *
     *  关键见解: 如果 MPS_BLOCK_COUNT == 32，你需要设置所有 32 位
     *  （0xFFFFFFFF）。如果 MPS_BLOCK_COUNT < 32，则只应设置低 N 位
     *  —— 未使用的高位必须保持 0，以便 __builtin_clz() 找到
     *  真实块，而不是虚幻的块。
     *
     *  提示: (1u << MPS_BLOCK_COUNT) - 1u 适用于 N < 32，
     *        但 N == 32 时会溢出。处理两种情况。
     */
    if (MPS_BLOCK_COUNT == 32) {
        pool->bitmap = 0xFFFFFFFFu;
    } else {
        pool->bitmap = (1u << MPS_BLOCK_COUNT) - 1u;
    }

    memset(pool->ref_count, 0, sizeof(pool->ref_count));
    memset(pool->storage, 0, sizeof(pool->storage));
    memset(pool->generation, 0, sizeof(pool->generation));
}
//修改前

void mps_init(MemPool_t *pool)
{
    // 一次清零整个结构体（如果没有后续赋值）
    memset(pool, 0, sizeof(MemPool_t));
  
    // 只设置 bitmap
    if (MPS_BLOCK_COUNT == 32) {
        pool->bitmap = 0xFFFFFFFFu;
    } else {
        pool->bitmap = (1u << MPS_BLOCK_COUNT) - 1u;
    }
}
//修改后
```
